# 🚀 Learning Hub

A comprehensive, interactive learning platform for modern DevOps technologies. Master Docker, Nginx, and CI/CD with hands-on tutorials, real-world examples, and interactive lessons.

## 📚 What's Inside

### 🐳 Docker Mastery
**Zero → Expert** | Complete containerization course

- **Foundations**: What is Docker, Installation, Images vs Containers, Essential Commands
- **Dockerfile**: Basics, Multi-stage Builds, Best Practices
- **Docker Compose**: Multi-container applications, Volumes, Networking
- **Production**: Security, Optimization, Real-world Spring Boot & React examples

**Key Features:**
- Interactive code editors with validation
- Terminal simulations
- Multi-stage build examples for Spring Boot and React
- Production-ready Docker Compose configurations
- Quizzes and hands-on challenges

### ⚡ Nginx Mastery
**Learn & Practice** | Web server and reverse proxy expertise

- **Fundamentals**: What is Nginx, Installation, Configuration Structure
- **Core Features**: Reverse Proxy, Load Balancing, SSL/HTTPS
- **Advanced**: Rate Limiting, Caching, Performance Tuning
- **Real World**: Full-stack React + Spring Boot deployment

**Key Features:**
- Live configuration validation
- Real-world proxy examples
- Load balancing simulations
- Complete production configs
- Security best practices

### 🔄 CI/CD Mastery
**Zero → Expert** | Continuous Integration & Deployment

- **Foundations**: Git workflows, Branching strategies
- **GitHub Actions**: Workflow anatomy, Build & Test pipelines
- **Docker Integration**: Build, Push, Deploy containers
- **Production**: Blue-green deployments, Rollback strategies
- **Real World**: Complete Spring Boot & React pipelines

**Key Features:**
- Interactive pipeline builders
- Deployment simulations
- Real GitHub Actions workflows
- Multi-environment strategies
- Rollback and recovery patterns

## 🎯 Learning Approach

### Interactive Learning
- **Live Code Editors**: Write and validate configurations in real-time
- **Terminal Simulations**: See command outputs without running actual commands
- **Quizzes**: Test your knowledge at each step
- **Hands-on Challenges**: Apply what you learned

### Real-World Examples
- **Demo App**: Spring Boot backend (Java 21, Gradle, Oracle DB)
- **React Frontend**: Modern SPA with Vite and TypeScript
- **Full Stack**: Complete deployment architecture
- **Production Ready**: Security, monitoring, and best practices

### Progressive Difficulty
- **Zero Level**: Complete beginners welcome
- **Beginner**: Core concepts and basic usage
- **Intermediate**: Advanced features and patterns
- **Expert**: Production optimization and troubleshooting

## 🚀 Getting Started

### Quick Start
1. Clone this repository
2. Open `index.html` in your browser
3. Choose your learning path
4. Start learning!

```bash
git clone <repository-url>
cd learning
# Open index.html in your browser
```

### No Installation Required
All courses run entirely in your browser. No Docker, Nginx, or CI/CD tools needed to learn!

### Recommended Learning Path

**For Backend Developers:**
1. Docker Mastery (Lessons 1-14)
2. Nginx Mastery (Lessons 1-18)
3. CI/CD Mastery (Lessons 1-17)

**For Frontend Developers:**
1. Docker Mastery (Focus on React examples)
2. Nginx Mastery (Focus on static file serving and SPA routing)
3. CI/CD Mastery (Focus on Node.js pipelines)

**For DevOps Engineers:**
Complete all three courses in order for comprehensive knowledge.
